# vision steering
A simple package for Gazebo which makes TurtleBot3 move towards red objects.

![alt text](image-1.png)
![alt text](image-2.png)

# Installation

Follow the installation guides for ROS and Gazebo, then run the following commands from your workspace:

'
mkdir -p ros2_ws/src
cd ros2_ws/src
git clone <your_repo>
cd ..
colcon build
'
